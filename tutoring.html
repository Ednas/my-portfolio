For Coding Bootcamp Students

I often refer to these links in my tutoring sessions. After having tutored students for a couple of years, I decided to curate them all within one page.

Full Stack Web Development

CSS

  • CSS Plates A game that teaches you about CSS selectors. You will be typing in selectors to select the correct plate or food item.
  • Flexbox Froggy A game similar to CSS Plates, this time you will be utilizing the flexbox framework to move three frogs to their color coordinated lily pad.

JavaScript

  • Free Code Camp You probably have been told my multitudes of people to check out this free resource, it is good, there are a lot of tutorials and explanations here. You can spend thousands of hours on this website. It can be a bit overwhelming.
  • Edabit challenges These bitesized challenges teach you how to write JavaScript code (available in other languages too!) by solving short challenges.
  • Scrimba These interactive screencasts help you understand a chunk of code and then let you solve the challenges.
  • Just JavaScript Dan Abramov's e-mail series
  • JavaScript Workbook Created by the curriculum engineer team for the Coding Bootcamp, this is a good overview of JS concepts and foundational knowledge.

  • Service Worker
